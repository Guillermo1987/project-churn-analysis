# 📊 Proyecto 1: Análisis Predictivo de Fuga de Clientes (Churn)

## Resumen del Proyecto
Este proyecto demuestra mi capacidad para aplicar técnicas de **Análisis de Datos** y **Machine Learning** para resolver un problema de negocio crítico: la retención de clientes. Utilizando un modelo de Regresión Logística, se identifican los factores clave que impulsan la fuga de clientes, permitiendo a los equipos de negocio implementar estrategias de retención proactivas.

Este proyecto conecta directamente mi experiencia en **SalesOps/RevOps** con mis habilidades técnicas en Data Science.

## Habilidades Demostradas
*   **Análisis Predictivo:** Implementación de un modelo de Regresión Logística.
*   **Programación:** Uso de Python (Pandas, Scikit-learn, Matplotlib, Seaborn).
*   **Visualización de Datos:** Generación de la Matriz de Confusión para evaluar el rendimiento del modelo.
*   **Preprocesamiento de Datos:** Manejo de variables categóricas y división de conjuntos de datos.
*   **Business Acumen:** Aplicación de Data Science para la optimización de la retención de clientes.

## Estructura del Repositorio
*   `churn_data.csv`: Conjunto de datos sintéticos de clientes utilizado para el análisis.
*   `churn_analysis.py`: Script principal de Python para el preprocesamiento, entrenamiento del modelo y evaluación.
*   `confusion_matrix.png`: Visualización de la Matriz de Confusión del modelo.
*   `analysis_results.md`: Informe detallado de los hallazgos clave y métricas de rendimiento del modelo.
